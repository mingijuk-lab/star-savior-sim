# 사이클 로테이션 마스터 (v8)

> **참조 순서**: 캐릭터 스펙 → `캐릭터_스펙_마스터.md` / 공식·규칙 → `시뮬레이션_가이드라인_v6.md`
>
> **업데이트 방법**: 캐릭터 로테이션 변경 시 해당 캐릭터 섹션만 수정.
> turns 배열 내 필드 의미는 **필드 가이드** 섹션 참조.

---

## 공통 필드 가이드

| 필드 | 타입 | 설명 |
|------|------|------|
| `coeff` | float | 스킬 계수 (1.20 = 120%) |
| `di` | float | 스킬 고유 피해량증가 합산 (DI) |
| `atk_buf` | float | 스킬 공격력% 버프 (AX: 곱연산 / 비AX: d_static 합산) |
| `cr_buf` | float | 치확% 버프 (cr_base에 가산) |
| `cd_buf` | float | 치피% 버프 (DI 합산에 영향) |
| `spd_mult` | float | 속도 배율 (1.00 = 버프없음, 1.20 = 속도+20%) |
| `is_ult` | bool | 궁극기 여부 → AX 스택 초기화 트리거 |
| `omega_elig` | bool | 오메가 DI 적용 여부 (특수기 계수>0인 턴만 true) |
| `is_basic` | bool | 기본기 여부 → EX 트리거 계산용 |

> **AX 스택**: turns 배열에 기재하지 않음. 코드에서 `ax_stacks[]`로 별도 추적.
> **힐데 전용 필드**: `coeff_atk`, `coeff_hp` 분리 (일반 `coeff` 미사용)
