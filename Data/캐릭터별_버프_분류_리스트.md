# 캐릭터별 버프/디버프 분류 리스트 (Standardized)

이 문서는 캐릭터별 효과를 **일반(Common)**, **공용 속성(Attribute)**, **고유(Unique)**로 분류하여 엔진 시뮬레이션의 중첩 로직 기준을 정의합니다.

---

## 1. 일반 버프/디버프 (Common)
*동일 효과군 내에서 미중첩, 최고 수치 우선 적용*

| 효과명 | 보유 캐릭터 | 표준 수치 예시 |
| :--- | :--- | :--- |
| **공격력 증가** | 린, 샤를(바니걸), 클레어, 유미나, 키라, 에핀델 | 30% |
| **방어력 감소** | 로자리아, 스마일 | 30% |
| **치명타 확률 증가** | 프레이, 클레어, 아세라 | 20~30% |
| **치명타 피해 증가** | 클레어 | 20% |
| **속도 증가** | 로자리아, 아세라, 에핀델, 레이시 | 15~20% |

---

## 2. 공용 속성 스택 (Common Attribute)
*클래스별 공통 시스템으로, 엔진 내부의 `attr_stack` 파이프라인으로 통합 연산*

| 명칭 | 해당 캐릭터 | 속성 (Mapping) |
| :--- | :--- | :--- |
| **도약 (Leap)** | 린, 벨리스, 스마일 | **별** |
| **격동 (Turmoil)** | 리디아, 레이시 | **혼돈** |
| **냉각 (Cold)** | 에핀델, 클레어(바니걸), 릴리 | **달** |
| **통찰 (Insight)** | 루나 | **질서** |
| **점화 (Ignition)** | 샤를(바니걸), 스칼렛(바니걸), 키라 | **태양** |

---

## 3. 고유 버프/상태 (Unique)
*일반/공용 스택과 독립적으로 중첩 합산 가능*

| 캐릭터 | 고유 명칭 | 주요 효과 |
| :--- | :--- | :--- |
| **린 (Lin)** | 명월 | 공격력 20% 증가 (일반 공증과 별개) |
| **린 (Lin)** | 청풍 | 속도 10% 증가, 특수기 쿨타임 감소 |
| **샤를 (바니걸)** | 럭키토큰 | 방어력 관통 20% (어쌔신) |
| **클레어 (바니걸)** | 잭팟 블룸 | **치확 +20%, 치피 +20% (3턴)** |
| **릴리 (Lily)** | 수사자문 | 치확 +15%. 보유 시 궁극기 피해 +30% |
| **로자리아 (Rosaria)** | 업화 | 특정 스택 도달 시 추가타 발동 및 DI 증가 |
| **프레이 (Frey)** | HR (High-Roll) | 스택 누적 시 기본기 피해량 증폭 |
| **유미나 (Yumina)** | 공명 | 고정 공격력 및 공격력 % 증가 (패시브) |
| **스칼렛 (바니걸)** | 세븐볼 | 개수당 주는 피해량 +7% 및 쿨타임 감소 |
| **레이시 (Lacy)** | 뒤엉킨 꿈결 | 적에게 부여. 해당 적 타격 시 피해량 +20% |
| **벨리스 (Belis)** | 사후 관리 | 아군 사망 시 공격력 증가 (최대 +30%) |

---

## 4. 엔진 구현 지침
*   **일반**: `common_buffs = {"ATK": 0, "DEF_RED": 0, ...}` 객체로 관리하여 중첩 방지.
*   **속성**: 사이클의 `attribute_stack` 혹은 비고란의 `도약+1` 등을 파싱하여 `attr_stack` 변수로 통합 관리.
*   **고유**: 캐릭터별 전용 지속턴/상태 변수로 개별 관리.
