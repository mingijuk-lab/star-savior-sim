# 스타 세이비어 — 데미지 시뮬레이션 가이드라인

> 본 문서는 AI에게 새로운 캐릭터 정보를 입력하여 데미지 시뮬레이션을 수행하기 위한 **종합 레퍼런스**입니다.
> 캐릭터 데이터를 입력 템플릿에 맞추어 제공하면, 이 문서의 규칙에 따라 DPS 계산·비교가 가능합니다.

---

## 1. 게임 기본 시스템

### 1.1 전투 구조
- **파티**: 4명 편성
- **턴제**: 행동력(게이지)이 100에 도달한 캐릭터가 1턴 행동
- **행동력 공식**: `턴당 행동력 증가 = 속도 × 0.1`
- 행동 후 게이지 0으로 초기화, 다시 속도에 따라 증가
- **행동 주기**: `1000 / 속도` (글로벌 턴 단위)

### 1.2 행동 게이지 증가 효과
- "행동 게이지 N% 증가" = 게이지가 100% 기준 N%만큼 즉시 증가
- 예: 행동 종료 후 게이지 0 → 행동 게이지 20% 증가 → 게이지 20에서 시작

### 1.3 스킬 우선순위
```
궁극기(쿨타임 완료시) > 특수기(쿨타임 완료시 또는 조건 충족시) > 기본기
```

### 1.4 쿨타임 규칙
- 쿨타임 N = 사용 후 N-1번의 **다른 행동**이 필요 (실질 N턴 주기)
- 예: 쿨타임 3 스킬 → 사용 후 2번의 다른 행동 → 4번째 턴에 재사용 가능 (T1 발동 ➡️ T2 대기 ➡️ T3 대기 ➡️ T4 발동)

### 1.5 추가 턴 (Extra Turn)
- 특정 스킬 사용 시 추가 턴 발생 → 턴을 넘기지 않고 즉시 추가 행동
- 추가 턴에서는 궁극기 사용 가능하면 궁극기 우선, 아니면 기본기 사용
- 버프 지속시간은 1턴만 소모된 것으로 계산
- 추가 턴에서 사용한 스킬도 쿨타임이 정상 적용됨

### 1.6 협동 공격
- 파티 내 협동 공격 능력 보유 캐릭터가 다른 캐릭터의 기본기 사용 시 확률적으로 기본기 발동
- 턴 소모 없음
- 협동 공격으로 발동한 기본기의 스택 효과도 정상 적용

## 2. 공격력 및 스탯 계산식

### 2.1 최종 공격력 공식
```
최종_공격력 = ( (ⓐ × (1 + ⓒ)) + 공명_정수 ) × (1 + ⓓ) + ⓑ
```

**각 항목 정의:**

| 기호 | 구성 요소 | 비고 |
|------|----------|------|
| **ⓐ** | 기본 공격력 + 1250 (스텔라 아카이브) | 캐릭터 시트의 공격력 + 1250 |
| **ⓑ** | 잠재력 + 호감도 + 전술분석 + 장비(정수) | **합산 약 1000으로 가정** |
| **ⓒ** | 공명 단계 퍼센트 | 캐릭터별 공명% (예: 6%) |
| **ⓓ** | 고정 공격력 퍼센트의 합연산 | 아래 상세 |

### 2.2 ⓓ 구성

```
ⓓ = 패시브_공격력%
    + 아르카나_공격력%
    + 여정_공격력%_기본 (AX 포함, 고정값 +8%)
    + 전술분석(1%)
    + 장비세트_공격력%
    + 장비_부옵션_공격력% (16.25% 고정)
    + [동적 버프: 리디아 패시브 스택%]
```

> **★ AX 여정 처리 규칙 (v6 확정):**
>
> | AX 구성 요소 | 값 | 적용 방식 |
> |---|---|---|
> | AX 기본 공격력 | +8% (고정) | ⓓ에 **합연산** |
> | AX 동적 스택 | +8% × N (최대 5회, 궁극기시 초기화) | 최종 공격력에 **곱연산** |
> | 스킬 공격력 버프 | 스킬별 상이 (예: 30%) | AX 동적 스택과 **합연산 후 곱연산** |
>
> **★ AX 여정 실효 공격력 공식:**
> ```
> eff_base = pool × (1 + ⓓ_정적) + ⓑ
> eff_atk  = eff_base × (1 + AX_스택 × 0.08 + 스킬_공격력_버프%)
> ```
>
> **비AX 여정 실효 공격력 공식:**
> ```
> eff_atk = pool × (1 + ⓓ_정적 + 스킬_공격력_버프%) + ⓑ
> ```

### 2.3 속도 계산
```
최종_속도 = 기본_속도 + 장비_부옵션(60) + 아르카나_속도 + 공명_속도 + [동적 스택]
```
- 장비 부옵션 속도: **60 고정**
- 동적 스택 (예: 어쌔신 턴시작 +10, 최대 3회) → steady-state에서 만스택 적용

### 2.4 치명타 계산
```
치명타_확률 = 기본_치확 + 아르카나_치확 + 여정_치확 + 장비_치확 + [패시브/버프 치확]
치명타_피해 = 기본_치피 + 여정_치피 + 장비_치피 + [패시브/버프/아르카나스택 치피]
```

## 3. 데미지 공식

### 3.1 기본 공식
```
데미지 = 실효_공격력 × 스킬_계수 × (1 + 피해량_증가 + 치명타_기여)
```

> **★ v6 핵심 변경: 치명타 피해는 피해량 증가(DI)와 합연산**
> 치명타는 별도 곱연산이 아니라 DI 합산 항목 안에 포함됩니다.

### 3.2 실효 공격력

**비AX 여정:**
```
eff_atk = pool × (1 + ⓓ_정적 + 스킬_공격력_버프%) + ⓑ
```

**AX 여정 (★ 핵심):**
```
eff_base = pool × (1 + ⓓ_정적) + ⓑ
eff_atk  = eff_base × (1 + AX_스택 × 0.08 + 스킬_공격력_버프%)
```

| 항목 | 내용 |
|------|------|
| `ⓓ_정적` | 패시브% + 리디아스택% + 아르카나% + **AX기본8%** + 전술1% + 장비세트% + **장비부옵16.25%** |
| `스킬_공격력_버프%` | 스킬에 의한 공격력% 버프 (AX 여정에서만 곱연산 그룹, 비AX는 ⓓ_정적과 합산) |
| `AX_스택 × 0.08` | 턴 시작 시 +1 스택 (최대 5 = +40%), 궁극기 후 초기화 |

### 3.3 피해량_증가 합산 (DI)
```
(1 + DI + 치명타_기여)

DI = 스킬_고유_피해증가 + 조건부_피해증가 + FX_carry + 특수기DMG_스택 + ...

치명타_기여:
  CR < 100%:  CR × CD     ← 기댓값
  CR ≥ 100%:  CD          ← 확정 치명타
```

> **모든 항목 합연산 (곱이 아닙니다)**
> 치명타 피해(CD)도 DI와 합연산되므로, CD 스탯의 한계 효율은
> 현재 CR 수준과 기존 DI 합산값에 의해 감쇄됩니다.

### 3.4 세트별 실질 효율 구조

치명타 합산 공식 하에서의 세트 효율 비교 (스트B, CR=35%, CD=50% 기준):

| 세트 | 기여 방식 | 실질 DPS 증가 |
|------|----------|-------------|
| **공격4세트 +20%** | eff_atk 직접 상승 | **~+11.8%** |
| **통찰4세트 +30%CR** | 치명타_기여 = +0.30×CD | ~+10.2% |
| **파괴4세트 +40%CD** | 치명타_기여 = CR×+0.40 | ~+9.4% |

> CR이 낮을수록 공격셋 우위, CR이 높을수록 세트 간 차이 축소

## 6. 여정 특수 잠재

| 잠재 | 기본 효과 | 동적 효과 | 적용 방식 |
|------|----------|----------|----------|
| **AX** | 공격력+8% (ⓓ 합연산) | 턴시작 공격력+8% (최대5회=+40%), 궁극기시 초기화 | **기본 +8% → ⓓ 합연산 / 동적 스택+스킬버프 → 곱연산** |
| **FX** | 치피+10% | 치명타시 다음 타격(계수>0)에 DI+25% | DI 합산 |
| **EX** | 치확+10% | 기본기 치명타시 25%확률 팀원 기본기 1회 발동 | 파티 DPS 별도 기여 |

### 6.1 AX 상세 규칙 (★ v6 확정)

1. **기본 +8%**: 여정 선택 시 고정 부여 → `ⓓ_정적`에 합연산
2. **동적 스택**: 턴 시작 시 +1 스택 (최대 5회)
3. **스킬 공격력 버프**: AX 동적 스택과 **합산 후 곱연산**
   ```
   eff_atk = eff_base × (1 + AX_스택 × 0.08 + 스킬_공격력_버프%)
   ```
4. **궁극기 사용 시**: 데미지 적용 **후** 스택 0으로 초기화 → 다음 턴 1스택부터 재시작
5. 사이클 내 턴별 스택 수를 반드시 추적해야 함

### 6.2 FX 상세 규칙
1. 타격(계수 > 0) 후 치명타 발생 시, 다음 타격의 DI +25% (합산)
2. 계수 0% 스킬(예: 세이라 특수기)은 타격이 아니므로 carry 소모하지 않음
3. **steady-state 가정**: 사이클 반복 시 마지막 타격의 carry가 다음 사이클 첫 타격에 전달
4. 기댓값: `이전 타격의 치확 × 25%`가 매 타격의 DI에 추가

### 6.3 EX 상세 규칙
1. 기본기로 치명타 발생 시, 25% 확률로 랜덤 팀원의 기본기 1회 발동
2. 자신의 턴에 1회만 발생
3. 기댓값: `기본기_횟수 × 치확 × 0.25 × 팀원_기본기_기댓값_데미지`
4. 팀원 기본기 대리값: `10,000(공격력) × 1.20(계수) × 1.15(피해증가) × 1.175(치명타보정) ≈ 16,215`

## 8. DPS 계산 구조

### 8.1 사이클 데미지
```
사이클_데미지 = Σ(각 행동의 데미지) + EX_보너스
```

### 8.2 DPS/전역턴
```
사이클_시간 = Σ(1000 / 각_턴_속도)   ← 속도 버프 턴별 반영
DPS = 사이클_데미지 / 사이클_시간
```
