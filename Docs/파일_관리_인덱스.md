## 상황별 업데이트 가이드

### ① 새 캐릭터 추가할 때

1.  **`Data/캐릭터_스펙_마스터.md`** → 맨 아래 "새 캐릭터 추가 템플릿" 복사하여 스탯·스킬 기입
2.  **`python Tools/extract_characters.py`** → JSON 데이터 동기화
3.  **`Data/사이클_로테이션_마스터.md`** → JSON 객체에 캐릭터 로테이션 추가 (혹은 `update_rotations.py` 실행)
4.  시뮬 실행: `python -m Core.calc_dps`

### ② 기존 캐릭터 수치 바뀌었을 때 (패치)

1.  **`Data/캐릭터_스펙_마스터.md`** → 해당 캐릭터 섹션 수치만 수정

### ③ 장비 부옵션(공, 속, 치확, 치피)을 바꿀 때

1.  **`python Core/calc_dps.py`** 실행 시 터미널 프롬프트에 원하는 수치 입력
2.  혹은 **`Data/equipments.json`**의 `base_pieces` 섹션을 수정 (고정 수치가 필요할 경우)

### ④ 새 장비 세트 효과 추가할 때

1.  **`Data/equipments.json`** → `four_piece_sets` 또는 `two_piece_sets`에 새 항목 추가
2.  시뮬레이션 시 자동으로 모든 4+2 조합이 생성되어 결과에 반영됨

## AI에게 시뮬 요청하는 방법

### 기본 요청

```
[캐릭터명]을 [아르카나] + [여정] + [장비세트] 조건으로 DPS 계산해줘.
```

예시:
```
로자리아를 레인저A + AX + 통찰4 조건으로 DPS 계산해줘.
유미나와 로자리아 전체 조합 비교 표 만들어줘.
```

### 새 캐릭터 시뮬 요청

```
스펙 마스터 파일에 [캐릭터명] 추가했어. 전체 여정×장비 DPS 비교해줘.
```
