# 스타 세이비어 시뮬레이션 — 파일 관리 인덱스

> 어떤 파일을 언제 열어야 하는지 한눈에 확인하는 가이드

---

## 파일 구조

```
📁 스타 세이비어 시뮬레이션
│
├── 📄 파일_관리_인덱스.md          ← 지금 보는 이 파일 (전체 구조 안내)
│
├── 📄 캐릭터_스펙_마스터.md        ← 캐릭터 기본 스탯 / 패시브 / 스킬 계수
│                                    "새 캐릭터 추가" 또는 "수치 수정" 시 편집
│
├── 📄 사이클_로테이션_마스터.md    ← 로테이션 순서 / turns JSON / AX 스택 배열
│                                    "로테이션 변경" 시 편집
│
└── 📄 시뮬레이션_가이드라인_v6.md  ← 데미지 공식 / 아르카나 / 여정 규칙
    (별도 제공)                       거의 변경 없음. 공식 참조용
```

---

## 상황별 업데이트 가이드

### ① 새 캐릭터 추가할 때

1. **`캐릭터_스펙_마스터.md`** → 맨 아래 "새 캐릭터 추가 템플릿" 복사하여 스탯·스킬 기입
2. **`사이클_로테이션_마스터.md`** → 맨 아래 "새 사이클 추가 템플릿" 복사하여 turns JSON 기입
3. 두 파일 모두 업로드 후 시뮬 요청

### ② 기존 캐릭터 수치 바뀌었을 때 (패치)

1. **`캐릭터_스펙_마스터.md`** → 해당 캐릭터 섹션 수치만 수정
2. 로테이션이 동일하면 사이클 파일은 수정 불필요

### ③ 로테이션 바뀌었을 때

1. **`사이클_로테이션_마스터.md`** → 해당 캐릭터 turns 배열 수정
2. 스탯이 동일하면 스펙 파일 수정 불필요

### ④ 새 장비·아르카나 추가될 때

- **`시뮬레이션_가이드라인_v6.md`** 장비/아르카나 섹션 수정 → 버전 번호 올리기 (v6 → v7 등)

## AI에게 시뮬 요청하는 방법

### 기본 요청 (파일 업로드 후)

```
[캐릭터명]을 [아르카나] + [여정] + [장비세트] 조건으로 DPS 계산해줘.
```

예시:
```
로자리아를 레인저A + AX + 통찰4 조건으로 DPS 계산해줘.
유미나와 로자리아 전체 조합 비교 표 만들어줘.
```

### 새 캐릭터 시뮬 요청

```
스펙 마스터 파일에 [캐릭터명] 추가했어. 전체 여정×장비 DPS 비교해줘.
```
