## 상황별 업데이트 가이드

### ① 새 캐릭터 추가할 때

1. **`Data/캐릭터_스펙_마스터.md`** → 맨 아래 "새 캐릭터 추가 템플릿" 복사하여 스탯·스킬 기입
2. **`Data/사이클_로테이션_마스터.md`** → JSON 객체에 캐릭터 로테이션 추가
3. 시뮬 실행: `python Core/calc_dps.py` → `python Core/generate_html.py`

### ② 기존 캐릭터 수치 바뀌었을 때 (패치)

1. **`Data/캐릭터_스펙_마스터.md`** → 해당 캐릭터 섹션 수치만 수정
2. 로테이션이 동일하면 사이클 파일은 수정 불필요

### ③ 로테이션 바뀌었을 때

1. **`Data/사이클_로테이션_마스터.md`** → 해당 캐릭터 turns 배열 수정
2. 스탯이 동일하면 스펙 파일 수정 불필요

### ④ 새 장비·아르카나 추가될 때

- **`Docs/시뮬레이션_가이드라인_v6.md`** 장비/아르카나 섹션 수정 → 버전 번호 올리기
- **`Core/calc_dps.py`** 내 `EQUIPMENTS` / `ARCANAS` 딕셔너리에 항목 추가

## AI에게 시뮬 요청하는 방법

### 기본 요청

```
[캐릭터명]을 [아르카나] + [여정] + [장비세트] 조건으로 DPS 계산해줘.
```

예시:
```
로자리아를 레인저A + AX + 통찰4 조건으로 DPS 계산해줘.
유미나와 로자리아 전체 조합 비교 표 만들어줘.
```

### 새 캐릭터 시뮬 요청

```
스펙 마스터 파일에 [캐릭터명] 추가했어. 전체 여정×장비 DPS 비교해줘.
```
