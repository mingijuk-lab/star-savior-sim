# 시각화 가이드 — optimization_guide.md → HTML 변환

> **목적**: `Results/optimization_guide.md` 마크다운 데이터를 인터랙티브 HTML 대시보드로 변환하기 위한 규격 문서입니다.

## 1. 아키텍처 개요

```
Results/optimization_guide.md  ──┐
                                 ├── generate_html.py ──► Results/optimization_guide.html
Core/calc_dps.py (main 함수)   ──┘
```

### 파이프라인
1. `Core/calc_dps.py`의 `main()` 함수가 시뮬레이션 실행 후 `optimization_guide.md`를 생성
2. `generate_html.py`가 마크다운을 파싱하여 JSON 데이터 구조로 변환
3. HTML 템플릿에 JSON 데이터를 삽입하여 단일 HTML 파일로 출력

## 4. 컴포넌트 상세

### 4.1 캐릭터 카드 (`char-card`)
- **접힌 상태**: 캐릭터명 + 배지 + 최고 DPS만 노출
- **펼친 상태**: 전략 탭, 순위 테이블, 빌드 진화 경로 노출

### 4.2 순위 테이블 (`rank-table`)
| 컬럼 | 스타일 |
|:---|:---|
| 순위 | `rank-num` (1=금, 2=은, 3=동) |
| 장비 세트 | `equip-tag` (모노스페이스, 다크 필) |
| 축복 | `bless-tag` (골드 보더) |
| 여정 조합 | `j-tag` (블루 보더, flex-wrap) |
| DPS | `dps-val` (녹색 JetBrains Mono) |
| MaxHit | `maxhit-val` (골드 JetBrains Mono) |

### 4.3 빌드 진화 경로 (`traj-section`)
- 스탯별(공격력%, 치피, 치확) 그룹
- 각 행: `pct-pill` + 장비명 + DPS + MaxHit + 여정 태그 리스트
- 중복 제거: 동일 장비/축복/여정이면 생략

### 4.4 No-Ult 전략 (`peak-box`)
- 빨간색 보더 박스
- 장비 + 축복 + 여정 + DPS + MaxHit 한 줄 노출

## 5. 인터랙션

| 기능 | 구현 |
|:---|:---|
| 캐릭터 검색 | `oninput` 이벤트로 카드 `hidden` 토글 |
| 카드 펼치기/접기 | 헤더 클릭 시 `open` 클래스 토글 |
| 전략 탭 전환 | `strat-tab` 클릭 시 패널 `visible` 토글 |
| 사이드바 내비게이션 | 클릭 시 `scrollIntoView` + 카드 자동 펼침 |

## 6. 자동 생성 파이프라인

### 실행 방법
```bash
# 1. 시뮬레이션 실행 (마크다운 생성)
python automated_run.py

# 2. HTML 변환
python generate_html.py
```

### generate_html.py 동작 흐름
1. `Results/optimization_guide.md` 파일 읽기
2. 정규식으로 캐릭터 섹션 분리 (`## 캐릭터명`)
3. 각 섹션에서 테이블 행 파싱 (Standard / No-Ult)
4. 빌드 진화 경로 파싱
5. JSON 데이터 배열 생성
6. HTML 템플릿에 삽입하여 `Results/optimization_guide.html` 출력
