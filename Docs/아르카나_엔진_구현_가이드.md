## 3. 엔진 수정 시 필수 체크리스트 (Audit Points)

1. **스택 캡(Cap) 확인**: 
    - 레인저는 5회, 캐스터/어쌔신은 3회, 오메가는 5회인지 상시 확인.
2. **트리거 시점 확인**:
    - `Turn-start`형(하늘의 심판 등)은 `any(j.name == ...)` 체크 후 즉시 가산.
    - `Action`형(깊은 애도, 오메가)은 `is_basic` 혹은 `is_spec` 조건부 가산.
3. **계수(Coefficient) 확인**:
    - 유미나 카탈리스트 5%, 오메가 5% 등 소수점 수치가 가이드와 일치하는지 확인.
4. **누락 방지**:
    - `multi_replace_file_content` 사용 시 수정 범위 밖의 인접 로직(예: 다른 if문)이 삭제되지 않도록 주의.
