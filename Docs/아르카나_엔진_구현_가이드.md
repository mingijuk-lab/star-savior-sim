## 3. 엔진 수정 시 필수 체크리스트 (Audit Points)

1. **스택 캡(Cap) 확인**: 
    - 레인저는 5회, 캐스터/어쌔신은 3회, 오메가는 5회인지 상시 확인.
2. **트리거 시점 확인**:
    - `Turn-start`형(하늘의 심판 등)은 `any(j.name == ...)` 체크 후 즉시 가산.
    - `Action`형(깊은 애도, 오메가)은 `is_basic` 혹은 `is_spec` 조건부 가산.
3. **계수(Coefficient) 확인**:
    - 유미나 카탈리스트 5%, 오메가 5% 등 소수점 수치가 가이드와 일치하는지 확인.
4. **고정밀 타이밍 (v15.1)**:
    - **AG 누적**: `ga_red_carry`는 항상 합산(`+=`) 처리하여 패시브와 JSON 보너스가 중첩되도록 함.
    - **추가 턴 (TN+1)**: `is_extra_turn: true`인 경우 소요 시간(`turn_time`)은 0.0이어야 하며, 행동 후 AG는 초기화됨.
5. **동적 전체 공격 판정 (v26.0)**:
    - **is_aoe**: `characters.json`의 각 스킬 "부가" 필드에 "전체 공격" 문자열이 포함된 경우 동적으로 활성화됨.
    - **피의 메아리 (Rosa EX)**: `is_aoe` 상태에서 치명타 발생 시 공격력 +3%(최대 5중첩) 버프 부여.
    - **경력직 용병 (Yumina EX)**: 해당 여정 장착 시 유미나의 **기본기(Basic)**를 `is_aoe`로 강제 판정하여 광역 효과 트리거 지원.
