# Star Savior DPS Simulation System

이 프로젝트는 스타 세이비어 캐릭터들의 성능을 다양한 장비, 아르카나, 여정 조합에 따라 시뮬레이션하고 분석하기 위한 도구입니다.

## 주요 기능
- **표준화된 행동 게이지(AG) 시뮬레이션**: 캐릭터의 속도와 행게증 패시브(유미나 15%, 프레이 30% 등)를 Tick 단위로 정밀하게 계산하여 실질 DPS를 산출합니다.
- **4인 파티 조합 시뮬레이터**: 개별 캐릭터의 성능을 넘어, 팀 전체의 행동 순서와 딜 지분을 분석하는 50턴 이상의 장기전 시뮬레이션을 지원합니다.
- **최적성 분석**: 수천 가지의 조합 중 각 캐릭터에게 가장 적합한 빌드를 추출합니다.
- **시각화 리더보드**: 계산 결과를 차트와 카드로 시각화하여 한눈에 비교할 수 있는 HTML 보고서를 생성합니다.

## 프로젝트 구조

```text
📁 Star/
│
├── 📁 Core/ (핵심 엔진 및 분석)
│   ├── 📄 calc_dps.py            ← DPS 시뮬레이션 메인 엔진 (AG 표준화)
│   ├── 📄 simulate_party.py      ← 4인 파티 조합 시뮬레이터 (신규)
│   ├── 📄 generate_html.py       ← 시각화 리더보드 HTML 생성
│   └── 📄 analyze_results.py     ← 결과 분석 및 요약 리포트
│
├── 📁 Data/ (마스터 데이터)
│   ├── 📄 캐릭터_스펙_마스터.md   ← 캐릭터 기본 스탯 및 스킬 계수
│   └── 📄 사이클_로테이션_마스터.md ← 스킬 순서 및 사이클 JSON 데이터
│
├── 📁 Docs/ (가이드라인)
│   ├── 📄 시뮬레이션_가이드라인_v6.md ← 행동 게이지(AG) 규칙 포함 가이드
│   └── 📄 파일_관리_인덱스.md      ← 파일 용도 및 관리 가이드라인 (v8.1)
│
├── 📁 Results/ (결과물)
│   ├── 📄 dps_results.csv         ← 시뮬레이션 로우 데이터
│   └── 📄 final_dps_report.md     ← 최신 정규 자동 합산 리포트
│
└── 📁 Tools/ (보조 도구)
    └── 📄 test_formulas.py       ← 데미지 수식 검증 테스트
```

## 실행 방법
1. **단일 DPS 시뮬레이션**:
   ```bash
   python Core/calc_dps.py
   ```
2. **4인 파티 조합 시뮬레이션**:
   ```bash
   python Core/simulate_party.py
   ```
3. **리더보드/리포트 생성**:
   ```bash
   python Core/generate_html.py
   ```

## 환경 요구사항
- **Python 3.x**: `pandas` 등의 라이브러리가 포함된 환경 권장
- **웹 브라우저**: 결과 시각화 확인용
